What is React?

React is an open source JavaScript library deployed by Facebook on May 29th, 2013. Reacts aim is to create faster UI for websites and apps. It is important to note that React is a library, not a framework. Reacts main focus is to create a fast and appealing UI for clients as easy as possible. Therefore vanilla React does not handle HTTP, routing, or many of the other backend capabilities a framework would.

Why should i learn react?

You may be reading this article and thinking "Why should i learn react, whats in it for me?". Well for starters React has been growing at an enormous rate since its release by Facebook. According to a Statista survey conducted in 2022, React had the most job oppurtunities over other frameworks and libraries such as Vue and Angular.
